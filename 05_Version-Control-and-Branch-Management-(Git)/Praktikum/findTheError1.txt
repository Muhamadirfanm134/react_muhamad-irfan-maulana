Git Flow Pak Dengklek:
[1] Create repository at github web
[2] git clone: repository_url
[3] git checkout -b development
[4] git pull
[5] git checkout -b feat/new-feature
[6] let's coding!
[7] git add .
[8] git commit -m 'feat(new-feature): add something crazy!'
[9] git push
[10] create pull request from feat/new-feature to development
[11] merge pull request
[12] confirm merge
[13] delete source branch
[14] create pull request from development to main
[15] merge pull request
[16] confirm merge
[17] delete source branch
[18] git checkout main
[19] git pull
